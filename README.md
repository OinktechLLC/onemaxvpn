# One Max VPN

**Production-ready аналог Samsung Max VPN**

- Браузерное расширение (Chrome/Edge/Firefox)
- Ежедневная смена серверов через поисковый робот
- Сжатие трафика, блокировка рекламы и трекеров

## Установка

1. Скачай ZIP из Releases
2. Распакуй `extension/` и загрузи в `chrome://extensions/`

## Ежедневное обновление серверов

GitHub Actions автоматически запускает `servers.py` каждый день.
