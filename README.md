# One Max VPN

**Аналог Samsung Max VPN (бывший Opera Max) — браузерное расширение для экономии трафика, приватности и блокировки рекламы.**

## Возможности
- Включение/выключение VPN одним кликом
- Прокси через `chrome.proxy`
- Блокировка трекеров и рекламы (declarativeNetRequest)
- Статистика сэкономленного трафика
- Ежедневная автоматическая смена серверов (поисковый робот)
- Адаптивный лендинг
- Автообновление расширения

## Установка

1. Скачай ZIP из [Releases](https://github.com/yourusername/one-max-vpn/releases)
2. Открой `chrome://extensions/`
3. Включи "Developer mode"
4. "Load unpacked" → выбери папку `extension`

## Структура проекта
- `extension/` — браузерное расширение
- `bot/` — поисковый робот для обновления серверов
- `landing/` — сайт для скачивания
- `.github/workflows/` — CI/CD

## Автообновление серверов
GitHub Actions запускает `bot/servers.py` каждый день в 00:00 UTC.
