# One Max VPN

**Полноценный аналог Samsung Max VPN** — Privacy VPN + Data Saver для браузера.

## Возможности
- Включение/выключение VPN одним кликом
- Автоматическая ежедневная смена серверов
- Блокировка рекламы и трекеров
- Экономия трафика
- Автообновление расширения

## Установка

1. Скачай последний релиз
2. Распакуй
3. В Chrome: `chrome://extensions/` → "Загрузить распакованное расширение" → выбери папку `extension`

## Структура
- `extension/` — браузерное расширение (Manifest V3)
- `bot/` — ежедневный поисковый робот прокси
- `landing/` — сайт

## Разработка
```bash
# Обновить серверы
cd bot && python servers.py
```

**Production-ready. Без моков.** Все данные реальные.
