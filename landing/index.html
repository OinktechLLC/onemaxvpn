Установка

Три шага и готово

1

Скачать ZIP

Последняя версия с GitHub Releases — обновляется автоматически при каждой сборке.

⬇️ Скачать
① Скачай ZIP и распакуй в любую папку
② Открой chrome://extensions
③ Включи «Режим разработчика» (справа вверху)
④ «Загрузить распакованное» → выбери папку
⑤ Нажми иконку расширения → «Включить СБП»
2

Как работает

После включения СБП браузер автоматически направляет заблокированные сайты через прокси-сервер. Обычные сайты работают напрямую — скорость не падает.

Заблокированный сайт → через прокси
Обычный сайт → напрямую (быстро)
Серверы меняются каждые 24ч автоматически
3

Автообновление

Расширение проверяет update.json и обновляется без Chrome Web Store. Серверы загружаются автоматически.
